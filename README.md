## 역할

- 그리기 기능
- 채우기 기능
- 만든 그림 다운로드
- 그림 지우기 기능
- 사진 추가하기 기능
- 지우개 기능
- 텍스트 작성 후 텍스트를 캔버스에 찍어내는 기능
- 색깔 선택 기능

## 배운점

- HTML5 canvas 사용법
- 2d context, 2d painting
- 이미지 저장 기능(a태그에 다운로드 속성이 있었다..)
